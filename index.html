Mad Metrics

Write a script that reports on several different aspects of the user's browsing behavior. You should be able to add this script to any web page. Since the metrics depend on scrolling, create a long, one-page landing page prototype to test this on. Add a button with position: fixed that the user can click to view the metrics collected so far. Metrics collection should be paused when this happens, and resumed afterwards. You can display the results in a manner of your choice (alert, lightbox, etc).
